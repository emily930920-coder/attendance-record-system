# 🚀 快速开始

## 三步启动

### 1️⃣ 安装依赖

```bash
npm install
```

### 2️⃣ 启动服务器

**方式一：使用npm命令**
```bash
npm start
```

**方式二：使用启动脚本**
- Mac/Linux: `./start.sh`
- Windows: 双击 `start.bat`

### 3️⃣ 打开浏览器

访问: http://localhost:3000

## 🎯 第一次使用

1. 看到界面后，点击"切换用户"
2. 输入你的姓名
3. 点击"上班打卡"开始记录

## 📱 手机访问

1. 确保手机和电脑在同一WiFi
2. 查看服务器启动时显示的局域网地址
3. 在手机浏览器中访问该地址

## 💡 快捷操作

- **打卡**: 点击对应按钮
- **查看统计**: 向下滚动页面
- **补录记录**: 点击"手动添加"按钮
- **导出数据**: 点击"导出数据"按钮

## ❓ 常见问题

**Q: 端口被占用怎么办？**
```bash
PORT=8080 npm start
```

**Q: 忘记打卡了？**
使用"手动添加"功能补录记录

**Q: 数据在哪里？**
服务器数据: `server/data/`
本地数据: 浏览器localStorage
