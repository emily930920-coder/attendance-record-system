## 🔄 服务器管理

### 查看运行状态
```bash
lsof -i:3000
```

### 停止服务器
```bash
# 找到进程ID
lsof -ti:3000

# 停止服务器（替换PID为实际的进程ID）
kill -9 [PID]
```

或者在运行服务器的终端按 `Ctrl + C`

## 🔧 常见问题

### Q: 端口被占用怎么办？

如果3000端口被其他程序占用，可以修改端口：

```bash
PORT=8080 npm start
```

然后访问: http://localhost:8080

### Q: 如何查看服务器日志？

服务器启动时的终端窗口会显示所有日志信息。

### Q: 忘记打卡怎么办？

使用"手动添加"功能补录记录：
1. 点击"手动添加"按钮
2. 选择日期和时间
3. 点击确定

### Q: 数据会丢失吗？

不会！数据同时保存在：
- 服务器文件系统（JSON文件）
- 浏览器本地存储（localStorage）

### Q: 如何导出数据？

点击"导出数据"按钮，会下载一个JSON文件。
